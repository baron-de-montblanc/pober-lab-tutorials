FHD/eppsilon pipeline on Oscar

The FHD/eppsilon pipeline is described in detail in Barry et al. 2019a.

Setting up IDL

To run FHD, we must first make sure we can run IDL on Oscar. CCV takes care of maintaining access to the license, but their documentation tends to fall out of date pretty fast (especially given the small amount of IDL users on Oscar). As of February 2025, here are the instructions to set up IDL on Oscar.

  1. Load the module via
    shell
    
$ module load idl
    This should set up the idl environment variable allowing you to start up the IDE.
  2. Run the following:
    shell
    
$ source /oscar/runtime/software/external/idl/idl89//bin/idl_setup.bash
    This takes care of correctly setting up the license.

If the setup is successful, running idl should return something like this:

shell
