FHD/eppsilon pipeline on Oscar

Summary

Your directory tree should now look like this:

  • fhd_deps/
    • coyote/
    • eppsilon/
    • FHD
    • fhdps_utils/
    • Healpix_3.83/
    • IDLAstro/
    • fhd_env.sh
    • idl_startup.pro
    • test_fhd.pro

Any time you want to run some IDL code, simply execute:

shell

$ source ~/fhd_deps/fhd_env.sh

If you ever want to double-check that all your FHD dependencies are correctly referenced, run:

shell

[~/fhd_deps]$ source fhd_env.sh
[~/fhd_deps]$ idl test_fhd.pro

Finally, if you end up needing more IDL dependencies for any reason whatsoever, simply download them and add their directory path to the existing idl_startup.pro file.
